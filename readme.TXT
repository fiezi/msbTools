This is the git repository for all Moviesandbox Tools based on OpenFrameworks.
These tools compile against openFrameworks 0.8 and most of them have project
files for Codeblocks.

You may this in any form but at your own risk.
msbKinect needs the ofxKinectOpenNI Addon
msbARSender needs the ofxARToolkitPlus Addon
msbMultiHIDInput supports Sixense Razor Hydra, I think...
